# synent-task2-Data-Visualization-Krina

[Problem Statement]:
The objective of this project is to perform Exploratory Data Analysis (EDA) on the Iris dataset to understand the distribution of features, identify relationships between variables, and gain insights into the characteristics of different iris flower species through data visualization.

[Dataset Details]:
The Iris dataset is a well-known dataset used for classification and data analysis tasks. It contains measurements of iris flowers from three species:
Iris-setosa
Iris-versicolor
Iris-virginica

Features:
Sepal Length (cm)
Sepal Width (cm)
Petal Length (cm)
Petal Width (cm)
Species (Target Variable)

[Approach]:
The analysis was carried out using Python libraries such as Pandas, Matplotlib, and Seaborn.

The following visualizations were created:
  Bar Chart – Shows the distribution of flower species in the dataset.
  Histograms – Analyze the distribution of Sepal Length and Petal Length.
  Scatter Plot – Examines the relationship between Sepal Length and Petal Length across different species.
  Pair Plot – Visualizes pairwise relationships among all numerical features.
  Correlation Heatmap – Identifies correlations between numerical variables.
These visualizations help uncover patterns, trends, and feature relationships within the dataset.

[Results]:
The analysis revealed clear differences among the three iris species based on their sepal and petal measurements. Petal-related features showed strong discriminatory power for species classification, while the correlation heatmap highlighted significant relationships between several numerical features.

Overall, the project provided valuable insights into the structure of the Iris dataset and demonstrated how visualization techniques can support data understanding before building machine learning models.
